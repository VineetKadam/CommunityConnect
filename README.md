
# Community Connect

**Community Connect** is a civic issue reporting platform designed to keep public and environmental concerns alive until justice is served. Users can raise awareness about civic, social, or global problems using media, real-time news, and public discussion â€” ensuring transparency, accountability, and long-term engagement, and contribute to a better community.


## Features

- Location-Based Reporting
- Admin Dashboard
- Whistleblower Mode(Anonymous Reporting)
- Public Sentiment Polls
- Community Discussion Threads
- Leaderboard
- Issue Timeline

## Tech Stack

**Client:** React.js | HTML | CSS | Tailwind CSS | JavaScript

**Server:** Node.js | Express.js | MongoDB | Mongoose
